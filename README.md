# 🃏 4x4 카드 뒤집기 기억력 게임 (Starter Project)

이 프로젝트는 초보자를 위한 **Python + Streamlit 기반의 카드 뒤집기 기억력 게임(Starter 프로젝트)**입니다.
복잡한 패턴이나 불필요한 파일을 최소화하고 `app.py`, `state.py`, `game_logic.py`로 역할을 분리하여 프로그램 구조가 단순하고 직관적으로 설계되어 있습니다.

## 🌟 프로젝트 소개
보이지 않는 상태의 열여섯 장(4x4 크기) 카드를 클릭하여 뒤집고, 두 장의 카드 그림(이모지)이 일치하면 오픈된 채로 짝을 완성하는 간단한 웹 게임입니다.
초보자 프로그래밍 교육용 템플릿 코드나 Streamlit의 세션 상태(`st.session_state`) 학습용으로 적합합니다.

## 🚀 실행 방법
이 프로젝트를 로컬 환경에서 실행하려면 아래 단계를 순서대로 진행해 주세요.

1. **Python 설치 확인**: Python 3.8 이상이 시스템에 설치되어 있어야 합니다.
2. **의존성 설치**: 터미널(명령 프롬프트)에서 해당 폴더로 진입 후, 다음과 같이 라이브러리를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
3. **앱 실행**: 아래 명령어를 통해 게임 애플리케이션을 구동합니다.
   ```bash
   streamlit run app.py
   ```
   명령어가 성공적으로 실행되면 기본 웹 브라우저가 실행되며 로컬 환경에서 게임이 시작됩니다.

## 📂 파일 구조
역할이 뚜렷하게 분리된 파일 구조를 가지고 있습니다.

- `app.py`: 웹 프론트엔드 UI를 렌더링하고 사용자 입력을 처리합니다.
- `game_logic.py`: 핵심 게임 로직(카드 초기화/섞기 기능 및 카드를 짝 맞추기 위해 비교하는 규칙)을 담당합니다.
- `state.py`: Streamlit이 다시 렌더링될 때 상태를 보존하기 위한 `st.session_state`를 초기화합니다.
- `requirements.txt`: 게임을 시작하기 위해 필요한 패키지만 명시되어 있습니다.
- `agent.md`: 본 프로젝트를 제작할 때 기준이 된 요구사항과 개발 원칙 등을 서술한 문서입니다.
- `README.md`: 프로젝트 안내 및 구성 설명을 위한 안내서입니다. (현재 문서)

## ✨ 현재 구현된 기능
학습과 이해도를 높이기 위해, 현재 버전에서는 아래 필수 기능에 한하여 구현되어 있습니다.

- [x] 4x4 사이즈의 카드 보드 렌더링
- [x] 총 8쌍의 그림 카드(이모지 16장) 무작위 섞기 패치 적용
- [x] 뒷면인 카드를 클릭할 시 즉시 뒤집어져 그림 확인 가능
- [x] 한 번에 두 장을 선택하여 동작 확인:
    - 선택한 두 장의 카드가 동일하면 해당 카드를 연 상태로 계속 유지
    - 내용이 일치하지 않는다면 플레이어가 확인할 수 있도록 약 1초간 보여준 뒤에 재차 닫기 모션 동작
- [x] 전체 카드를 성공적으로 매칭하면 축하 메시지 상자와 축하 풍선 이펙트 노출

## 💡 확장 아이디어
위와 같은 기초 코드를 분석하여 이해했다면, 자유롭게 다음과 같은 기능을 추가해 보세요.

- **타이머 표시**: 게임 진입 시간과 클리어 시간 사이의 경과 시간을 확인하는 실시간 제한 시간 기능 추가
- **점수 및 횟수 시스템**: 카드를 뒤집을 때마다 시도 횟수를 누적시키고, 가장 적게 걸린 사람의 기록 저장을 하도록 기능 추가
- **크기 및 난이도 조절**: 하드코딩된 크기가 아닌 4x4, 6x6 보드 사이즈 등 난이도 셀렉트박스를 이용한 동적 난이도 선택 지원 기능 추가
- **2인 대전 모드**: 사용자 턴을 도입해 점수를 따로 관리하고 누가 가장 많이 맞췄는지 스코어보드를 비교하는 기능 확장
